# 2026-07-21 覓投AI v0.2.0 开发日志

## 用户系统 + 订阅套餐 + 推送通知上线
- 实现 JWT 认证（注册/登录），密码 bcrypt 哈希
- 三个订阅套餐：免费版(3次/天)、专业版(50次/天/¥39月)、大师版(无限/¥99月)
- 混合模式C：免费用平台key，大师版可自带Key(BYOK)
- 配额系统：每日自动重置，超量返回429
- PushPlus 微信推送集成：分析完成即时通知
- 应用内通知历史记录
